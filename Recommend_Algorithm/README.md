### #2. 간단한 추천시스템 구현해보기
 - 총 4가지의 방법을 사용하여 평점 예측
 - 방법에 대한 성능평가는 RMSE 값 사용
   + 단순랜덤예측           : 1.92
   + 영화평균 평점기반 예측 : 1.03
   + 사용자평균 평점기반 예측 : 0.94
   + 장르별 평점기반 예측 : 1.06
 - 사용자 평균 기반으로 한 예측모형이 가장 RMSE값이 낮게 나타났다.
 - 즉, 사용자를 중심으로 모델링을 해야 성능이 향상 됨을 간단하게나마 확인할 수 있었다.
